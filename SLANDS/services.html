BOSS — Business Opportunity Software Search

Software that searches and filters recently sold homes to filter properties most likely to need remodeling. It helps local service businesses (e.g., roofing) target high-intent leads right after a purchase.

Goal: Demonstrate the type of work we can deliver that can add real value to local businesses.

    Key Functionalities
  • Pulls recent sales data and normalizes by neighborhood & price band
  • Scoring model ranks homes by "remodel likelihood"
  • Filters by city, price, beds/baths, and close-date window
  • Exports ranked lead lists

Tech used: Python • React • PostgreSQL • APIs
